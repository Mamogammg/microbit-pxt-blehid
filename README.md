# i2c2

```package
pins=github:bsiever/microbit-pxt-i2cpins
```

This extension allows non-default pins to be used for the micro:bit's excernal I2C.
# Setting the pins

```sig
pins.setI2CPins(sda : DigitalPin,  scl : DigitalPin) : void
```

Set the Data and Clock pins.
